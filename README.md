# Mark 美股智能戰情室

Real-time stock monitoring dashboard for US stocks with technical analysis signals.

## Features

- Multi-timeframe analysis (10m, 15m, 30m, 1h, 2h, 3h, 4h, 1d)
- Custom trading strategy signals
- Auto-refresh capability
- Sector-based organization
- Dynamic ticker management

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

## Data Source

Yahoo Finance via yfinance
